# gson-apt

> gson-apt使用android编译时生成代码(APT)自动生成Gson TypeAdatper代码，提供高性能解析json
> 两种模式，第一种极速模式，所有类都递归生成，但是生成代码比较多

## WHAT

### JSON值类型对应java类型

|ID|JSON value值类型|java类型|
|:-:|:-:|:-:|
|1|数值(number)|int、long、float、double|
|2|字符串（string）|String|
|3|true、false|boolean|
|4|对象(object)|Object|
|5|数组(array)|Object[]、List\<Object\>|
|6|null|各个类型的默认值|
